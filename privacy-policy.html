Privacy Policy

Last updated: July 2, 2026

GA: Re-run with Inputs is a Chrome extension that adds a "Re-run with new inputs" button to GitHub Actions run pages, allowing you to re-trigger workflows with pre-filled and modified inputs.

Data We Collect

We do not collect, transmit, or store any personal data on external servers. The extension operates entirely within your browser.

Data Stored Locally

The extension stores the following data in Chrome's local storage (chrome.storage.local) on your device only:

  • GitHub Personal Access Token (PAT) — optionally provided by you to authenticate with the GitHub API for private repositories. This token never leaves your browser and is never sent to any server other than api.github.com.

This data remains on your device and is never shared with third parties. You can remove it at any time by clicking the extension icon and using the "Remove saved token" button.

External Services

The extension communicates only with the following services on your behalf:

  • github.com — to read workflow YAML files and job logs using your active GitHub session
  • api.github.com — to fetch workflow run details and trigger new workflow runs using your PAT
  • GitHub log storage (e.g. *.actions.githubusercontent.com) — to retrieve job log files for reading previous run inputs

All requests are made directly from your browser to these services. No data passes through any intermediate server.

Permissions

The extension requests the following browser permissions:

  • storage — to save your PAT locally in Chrome
  • Host access to github.com and api.github.com — to read workflow files and interact with the GitHub API

No Analytics or Tracking

The extension contains no analytics, crash reporting, advertising, or any form of usage tracking.

Changes to This Policy

If this policy changes, the updated version will be published here with a new "Last updated" date.
